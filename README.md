## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/woowa-techcamp-2022/web-kiosk-yeonjeong.git
   ```
2. Install NPM packages
   ```sh
   npm install

   cd client
   npm install

   cd server
   npm install
   ```
3. DB configuration (`/server/.env`)
   ```
   MYSQL_HOST=
   MYSQL_USER=
   MYSQL_PW=
   MYSQL_DB=
   ```
